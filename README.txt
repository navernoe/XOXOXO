
>>> УСТАНОВКА <<<

 Чтобы установить игру, необходимо запустить на выполнение файл game_setup.sql в psql таким образом:
(используйте один из 3х вариантов)
1) \i <абсолютный_путь_до_файла>/game_setup.sql
2) \ir <относительный_путь_до_файла>/game_setup.sql
3) \cd <путь_до_файла>/
	\ir game_setup.sql


>>> ИНТЕРФЕЙС <<<

---> Чтобы начать новую игру заново: SELECT * FROM rungame();

---> Чтобы сделать ход: SELECT * FROM go(tr, td);
где tr - номер строки (integer), td - название столбца (char) (a,b или c)
ПРИМЕР: SELECT * FROM go(1, 'a');

--> Можно также отключить режим PVE, для этого в rungame() нужно передать параметр false,
чтобы включить обратно - true
ПРИМЕР: SELECT * FROM rungame(false);

--> В режиме Player vs Player второй игрок должен ходить, указывая 3м параметром символ 'O', т.е так:
SELECT * FROM go(1, 'a', 'O');
